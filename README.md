Dieses kleine Paket liest die Berechtigungen auf Ordnern (und sogar Dateien auf Wunsch) aus, listet alle Sicherheitsgruppen, User und deren Berechtigungsstatus.

Die Tiefe der durchsuchten Zweige kann angegeben werden. 

Zur Ansicht dient eine schlanke html-Datei mit Suchfilter.



This small package reads the permissions on folders (and even files if desired), lists all security groups, users and their permission status. 
The depth of the searched branches can be specified. A slim html file with a search filter is used for viewing.
